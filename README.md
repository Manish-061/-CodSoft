# Description
This project is a landing page for a business consultancy firm. It showcases the company's services, client testimonials, and provides a contact form for potential clients.

- CSS3

# Responsiveness

- Desktop: Full layout with all sections visible

# Task 3
# Simple Calculator

# Description
This project is a simple, yet stylish calculator web application. It provides basic arithmetic operations and a clean, responsive user interface.

# Features
- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Clear function to reset the display
- Keyboard support for input
- Responsive design
- Error handling for invalid calculations

# Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

# Project Structure
- `index.html`: Contains the structure of the calculator
- `styles.css`: Defines the styles for the calculator interface
- `script.js`: Implements the calculator's functionality

# Usage
- Click on the buttons to input numbers and operators
- Press '=' or click the equals button to calculate the result
- Press 'C' or click the clear button to reset the display
- You can also use your keyboard for input:
  - Numbers 0-9 for digits
  - '+', '-', '*', '/' for operations
  - 'Enter' for equals
  - 'Backspace' or 'Delete' for clear

# Customization
To customize the calculator's appearance:
1. Modify the colors in the `styles.css` file
2. Adjust the button sizes or layout in both `styles.css` and `index.html`

# Browser Compatibility
This calculator should work in all modern browsers that support ES6+ JavaScript.

# Future Improvements
- Add more advanced operations (square root, exponents, etc.)
- Implement a history feature to review past calculations
- Add a toggle for scientific calculator mode
